# Stories, Systems, and Software

This is a personal portfolio website created for my final project. The site highlights my professional background, technical skills, and selected projects.

## Project Concept

The theme of this website is "Stories, Systems, and Software." I wanted the design to connect my interests in computer science, storytelling, research, and web development.

## Features

- Responsive navigation
- About section
- Work and skills section
- Projects section
- Contact form
- Dark mode / reading mode toggle
- Project filtering
- Expandable project details
- Form validation with JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages
