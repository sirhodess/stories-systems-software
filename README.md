# Stories, Systems, and Software

Stories, Systems, and Software is a personal portfolio website created for my final web development project. The site highlights my background in computer science, selected software projects, technical skills, and interest in human-centered applications.

## Project Concept

The theme of this website is Stories, Systems, and Software. For this project, I wanted to create a portfolio that connects my professional background, academic work, and interest in building software that is clear, useful, and accessible.

The site focuses on my growing interests in human-computer interaction, EdTech, accessibility, voice user interfaces, and practical software tools. Instead of only presenting projects as technical work, the portfolio also considers how people interact with software and how design choices can make applications easier to understand and use.

My goal was to build a clean, responsive portfolio that presents both my technical skills and my interest in creating thoughtful software for learning, creativity, and real-world workflows.

## Features

- Responsive navigation for desktop and mobile screens
- About section with academic and professional background
- Work and skills section
- Projects section with selected technical work
- Project filtering by category
- Expandable project details
- Dark mode / reading mode toggle
- Animated canvas background with falling code
- Contact form
- JavaScript form validation
- Clean layout with readable spacing and accessible design choices

## Technologies Used

- HTML
- CSS
- JavaScript
- HTML Canvas
- GitHub Pages

## How to Run the Project

To run the project locally:

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.

No additional installation is required.

## Outside Sources

The following outside sources and references were used or consulted during the creation of this project:

### MDN Web Docs — HTML canvas Element

Used as a reference for understanding how the canvas element works and how it can be used to draw graphics with JavaScript.

### MDN Web Docs — Canvas API / CanvasRenderingContext2D

Used as a reference for drawing text and graphics on the canvas, including methods such as getContext(), fillText(), and canvas sizing.

### MDN Web Docs — requestAnimationFrame()

Used as a reference for creating smooth browser-based animation for the falling code background.

### MDN Web Docs — JavaScript DOM Methods

Used as a reference for working with elements on the page, including querySelector(), addEventListener(), classList, and updating page content with JavaScript.

### MDN Web Docs — Web Storage API / localStorage

Used as a reference for saving the user’s selected theme so the dark mode / reading mode preference remains active after refreshing the page.

### MDN Web Docs — HTML Form Validation

Used as a reference for validating contact form fields with JavaScript.

### GitHub Pages Documentation

Used as a reference for deploying the portfolio website online.

### Course Materials and Class Examples

Used as general guidance for HTML structure, CSS styling, responsive layouts, and JavaScript interactivity.
